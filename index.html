What it does

One product for the actual loop.

Most AI tools stop at generation. Nodes Nodes Nodes keeps setup, execution, inspection, and review inside one local workspace so the project stays legible from first prompt to final output.

Infinite canvas workflows

Connect prompt notes, lists, templates, assets, and model nodes on one graph.

Text, data, images, video, files

Work across mixed inputs and outputs without flattening everything into one chat transcript.

Local tools + provider keys

Use your own keys, add local tools, and keep everything inside the workflow.

Debug, inspect, iterate

See each run, review the results, and keep moving without losing the thread.

Quick answers

Questions that matter before you try it.

What is Nodes Nodes Nodes?

A local-first Mac app for building, running, debugging, and reviewing AI workflows on an infinite canvas.

Is it available right now?

Yes. It is a very early macOS alpha, available through the GitHub repo.

What does it work with today?

OpenAI, Google Gemini, and Topaz.

Does it run in the browser?

No. It is a desktop app.

Does it support collaboration or cloud sync?

Not yet. Today it is built for one person working locally.
