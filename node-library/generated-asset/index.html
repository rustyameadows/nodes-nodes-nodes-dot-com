Generated Asset Block

What it does

  • Preserves outputs inside the same project and flow.
  • Makes variations easy to compare.
  • Lets strong results feed the next step.

Use it when

  • A run produces something worth keeping.
  • You want to compare multiple results side by side.
  • The output should become source material for another pass.

Works well with

Model blocks, uploaded assets, and review views.

Best for

Saved outputs, variations, and next-pass source material.

Why it stays on the canvas

The output remains connected to the work that made it.
