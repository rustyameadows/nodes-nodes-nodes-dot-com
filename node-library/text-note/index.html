Text Note Block

What it does

  • Holds prompt language in a form you can edit quickly.
  • Keeps human context visible next to the rest of the flow.
  • Gives the canvas a clear written anchor.

Use it when

  • You are shaping a prompt before a run.
  • You want to keep instructions or reference wording close.
  • A project needs a human note that should not disappear into a side panel.

Works well with

Model blocks, templates, and lists.

Best for

Prompt language, directions, working notes, and context that needs to stay visible.

Why it stays on the canvas

It keeps the human thinking visible instead of hidden inside a side tool.
