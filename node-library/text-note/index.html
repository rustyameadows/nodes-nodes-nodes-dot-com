Text Note

The Text Note block is the simplest piece on the canvas: plain words that stay close to the rest of the work.

Back to library Get the early alpha
Text Note placeholder

What it helps you do

  • Write prompts, reminders, and instructions in normal language.
  • Keep useful wording visible while you build around it.
  • Give the rest of the project a clear written starting point.

When you would use it

  • When you have an idea you want to capture before it slips away.
  • When a model block needs cleaner directions.
  • When a project needs a plain-language note that is easy to revisit later.

Often used with

Model blocks, templates, and lists.

Good for

Prompts, project notes, quick instructions, and little pieces of context you do not want to lose.

Why it matters

Sometimes the best way to keep work moving is still a simple note in the right place.
