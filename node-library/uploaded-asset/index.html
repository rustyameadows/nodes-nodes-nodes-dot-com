Uploaded Asset

The Uploaded Asset block is for real source material you bring into a project and want to keep in view.

Back to library Get the early alpha
Uploaded Asset placeholder

What it helps you do

  • Bring a real image or file into the same project space as the rest of your work.
  • Keep source material easy to find while you build around it.
  • Use existing material as the starting point for what comes next.

When you would use it

  • When you already have a source image you want to work from.
  • When a project needs reference material close at hand.
  • When you want existing assets to sit beside the prompts and notes that matter to them.

Often used with

Model blocks, text notes, and generated assets.

Good for

Reference images, source material, and project files you want visible while the work evolves.

Why it matters

Source material stays part of the conversation instead of disappearing into a folder somewhere else.
