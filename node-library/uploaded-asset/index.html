Uploaded Asset Block

What it does

  • Keeps reference images and source files visible on the canvas.
  • Makes real project material part of the flow.
  • Gives model blocks something concrete to work from.

Use it when

  • You already have an image or file that matters to the run.
  • A project needs reference material close at hand.
  • You want source material tied to the exact flow using it.

Works well with

Model blocks, text notes, and generated assets.

Best for

Reference images, source material, and project files that need to stay visible.

Why it stays on the canvas

The source remains part of the story instead of disappearing into another folder.
