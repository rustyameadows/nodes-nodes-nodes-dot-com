Template

The Template block is for wording you plan to reuse, with a few parts that can change from one pass to the next.

Back to library Get the early alpha
Template placeholder

What it helps you do

  • Keep repeatable wording in one clean place.
  • Change a few parts without rewriting the whole thing every time.
  • Bring more consistency to work that happens again and again.

When you would use it

  • When a project needs the same prompt shape more than once.
  • When you want to reuse a format but swap in new details.
  • When you are tired of fixing the same wording by hand.

Often used with

Text notes, lists, and model blocks.

Good for

Repeatable prompts, naming patterns, and consistent directions that should stay stable across a project.

Why it matters

A little repeatability can save a lot of cleanup later.
