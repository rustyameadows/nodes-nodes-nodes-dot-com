What it helps you do

  • Run a prompt or source image through a provider you already use.
  • Create new writing, images, or transformed outputs without leaving the project.
  • Keep the result tied to the work that led up to it.

When you would use it

  • When you are ready to ask for a first pass or a variation.
  • When you want an existing image to become the next thing.
  • When you want the app to turn your setup into new material to keep working with.

Often used with

Text notes, templates, lists, and uploaded assets.

Good for

Image generation, image changes, and new written material you want to shape further.

What it leaves behind

A run you can check later and results that stay connected to the same project.
