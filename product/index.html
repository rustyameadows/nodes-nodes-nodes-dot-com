Product

A local-first Mac app for serious workflow tinkerers.

Build multimodal workflows on an infinite canvas, run jobs through your own provider keys, inspect queue and request-response detail, and review outputs locally without splitting the work across six tools.

Stylized placeholder art echoing the real dark-canvas workflow view.

Workspace model

A local-first Mac workspace

Projects stay isolated on disk, metadata stays local, and the app keeps prompts, nodes, assets, and runs attached to the same workspace.

Provider model

Bring your own providers

Use your own OpenAI, Google Gemini, and Topaz credentials, with queue activity and request-response detail visible in the app.

Capabilities

Built for the messy middle of real work.

Infinite canvas workflows

Build with notes, templates, assets, lists, and model nodes inside one connected graph.

Isolated local projects

Keep files, metadata, outputs, and project state together instead of in one global history pile.

Provider-agnostic execution

Run image and text jobs from one surface, wire in local tools, and keep your own provider keys in control.

Debug and review

Inspect queue state, provider request-response detail, and outputs in local compare views.

App Home for creating and reopening isolated local projects.

Asset review for rating, comparing, and keeping the outputs worth another pass.

Status

Very early alpha, with the repo as the current handoff.

The current build is an unsigned Apple Silicon macOS alpha. The GitHub repo is the right place for install notes, source-run fallback, and rough-edge expectations.
