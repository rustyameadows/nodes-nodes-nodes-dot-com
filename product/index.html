Product

A local-first Mac workspace for multimodal AI workflows.

Build on an infinite canvas, run jobs through your own provider keys, see runs and provider detail, and review outputs locally without splitting the work across multiple products.

Workspace

A local-first Mac workspace

Projects stay isolated on disk, metadata stays local, and the app keeps prompts, nodes, assets, and runs attached to the same workspace.

Providers

Bring your own providers

Use your own OpenAI, Google Gemini, and Topaz credentials, with runs and request details visible in the app.

Capabilities

Built for the parts most tools hide.

Infinite canvas workflows

Build with notes, templates, assets, lists, and model nodes inside one connected graph.

Isolated local projects

Keep files, metadata, outputs, and project state together instead of in one global history pile.

Provider-agnostic execution

Run image and text jobs from one surface, wire in local tools, and keep your own provider keys in control.

Debug and review

See runs, request details, and outputs in local compare views.
