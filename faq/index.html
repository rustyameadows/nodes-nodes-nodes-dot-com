General

What the product is

What is Nodes Nodes Nodes?

A local-first desktop app for building, running, and reviewing creative AI workflows.

Who is it for?

Creative AI power users who want more structure than scattered provider-specific interfaces.

What makes it different?

It combines a node canvas, supported provider runs, and comparison-oriented review in one app.

Platform and availability

Where it runs and what stage it is in

Is it available right now?

It is in macOS alpha and still being prepared for broader access.

Does it run in the browser?

No. The current product is a desktop app.

Does it support Windows or Linux?

Not in the current public site scope. The outward-facing messaging is macOS first.

Can I download the alpha today?

No public alpha download is posted yet.

Workflow and providers

What the current product covers

What providers does it support?

The current product docs describe OpenAI and Topaz support in the active workflow.

What can I do in the app?

Create projects, build workflow graphs, run supported provider jobs, and review outputs in asset views.

Does it support collaboration or cloud sync?

Not in the current version. The product is local-first and single-user today.
