FAQ

Questions before you try it.

The fast read on what the product is, who it is for, what works today, and what is still early.

Product

What it is and why it exists.

What is Nodes Nodes Nodes?

A visual workflow canvas for creative AI work on macOS.

Who is it built for?

People who work across prompts, source material, variants, and review and want more structure than a stack of provider tabs.

What makes it different from provider UIs?

It treats generation as a connected workflow, with a canvas, visible jobs, and review inside the same product.

What can I place on the canvas?

Text notes, lists, templates, uploaded assets, model blocks, and generated outputs.

Availability

Where it runs and how early it is.

Is it available now?

Yes. The current alpha is handed off through GitHub.

Is it Mac only?

Right now, yes. The current alpha is focused on Apple Silicon Macs.

Does it run in the browser?

No. It is a desktop product.

Do I need an account?

No. The current alpha is built for one local user working on one Mac.

Providers and runs

What works today.

Which providers work today?

OpenAI, Google Gemini, and Topaz.

How do provider keys fit in?

You bring your own keys. The app is designed to work with the providers you already use.

What is the Node Library for?

It is the guide to the canvas block system. Use it to understand what each block does before you start wiring a project.

Can I inspect what happened in a run?

Yes. Job state stays visible, and the app keeps a clear view of what happened when a run moves, succeeds, or fails.

Limits right now

What is still early or missing.

Does it support collaboration or cloud sync?

Not yet. The current product is local-first and single-user.

Is this a polished public release?

No. It is a real alpha, not a mass-market release.

Does local-first mean my work stays on my Mac?

Yes. Projects, canvas state, and outputs are built around a local desktop workflow.

Where should I go for privacy details?

Start with the privacy page. It holds the current plain-language overview while the fuller policy is being finalized.
